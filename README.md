# programacion2-segundaparcial

Programación 2 es una aplicación móvil para Android desarrollada como parte del segundo parcial de la materia.
La funcionalidad principal de la aplicación gira en torno al uso de la cámara del dispositivo para capturar imágenes y guardarlas en una galería interna.
El proyecto está construido utilizando tecnologías modernas de Android y sigue las mejores prácticas recomendadas por Google.

Funcionalidades Principales

Captura con Cámara: La aplicación solicita los permisos necesarios para acceder a la cámara y permite al usuario tomar una fotografía.

Navegación Intuitiva: Se utiliza Android Jetpack Navigation Component para gestionar el flujo entre las diferentes pantallas de la aplicación, garantizando una experiencia de usuario fluida y consistente.

Visualización de Contenido: Una vez capturada la imagen, se muestra en una pantalla de vista previa y se almacena localmente para su posterior visualización en la galería.
